Seven-Segment OCR Add-on

Enter the path of the image saved by Home Assistant (e.g. /share/cam.jpg).

Instructions: Click 3 points on the image to define a digit's bounding box:

  1. Lower-Left Corner (P)
  2. Lower-Right Corner to define X vector
  3. Upper-Left Corner to define Y vector
You can define multiple digits. When done, click "Detect".
